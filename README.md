### Getting started

#### Step 1: Install Apto-UI
##### NPM
```
npm install --save apto-ui
```
##### Yarn
```
yarn add apto-ui
```

#### Step 2: Import the component modules
```
import { AptoButtonComponentModule } from 'apto-ui';

@NgModule({
    ...
    imports: [
        AptoButtonComponentModule
    ],
    ...
})
export class SuperAwesomeAppModule { }
```
